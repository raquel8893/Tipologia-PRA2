4.2 Normalidad y homogeneidad de la varianza¶

Test de normalidad¶

Para comprobar la normalidad de las variables numricas Age y Fare aplicaremos el test Shapiro-Wilk. La hiptesis nula es que la variable sigue una distribucin normal. El nivel de significancia ser $\alpha = 0.05$, si el p-valor es inferior a este, rechazaremos la hiptesis nula.

La distribucin de la media de las variables estudidas, gracias a la gran cantidad de observaciones, seguir una distribucin normal con una media de poblacin $\mu$ y una varianza $\frac{\sigma^2}{N}$.

Test de homocedasticidad¶

Para comprobar la homocedasticidad (igualdad de varianzas entre los grupos que se van a comparar) de las variables usaremos el test de Levene, ya que debido a la gran cantidad de observaciones consideramos que nuestros datos siguen una distribucin normal. La hiptesis nula asume igualdad de varianza.

En este caso, podremos contrastar la diferencia de medias ya que los tamaos de las muestras son superiores a treinta. El teorema del lmite central nos dice que si los tamaos de las muestras son superiores a 30, el estadstico de contraste es una observacin de una variable aleatoria que se distribuye aproximadamente como una N(0,1).

$H_0: \mu_1 = \mu_2 $
